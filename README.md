# ML-LAB-10
Lab 10: Iris Dataset & SVM
This is a quick lab notebook where I train a Support Vector Machine (SVM) to classify the Iris dataset.

What I did in this lab:
EDA: Loaded the data using Seaborn and visualized the features with pairplots and KDE plots.

Modeling: Trained a baseline SVM model to classify the flower species.

Tuning: Used GridSearchCV to test different C and gamma hyperparameters and find the best combination.

Evaluation: Checked the final model's performance using a confusion matrix and classification report.

Libraries you need:
pandas, numpy, matplotlib, seaborn, scikit-learn
